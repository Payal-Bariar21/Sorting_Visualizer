# Sorting_Visualizer
An interactive web application that visualizes popular sorting algorithms through real-time animations. Users can generate random arrays, adjust sorting speed, and compare the behavior of different sorting techniques.

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
- Tailwind CSS
- Vite

## Features
- Visualizes Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, and Quick Sort
- Adjustable array size and animation speed
- Real-time sorting animations
- Responsive user interface
- Educational tool for understanding sorting algorithms
